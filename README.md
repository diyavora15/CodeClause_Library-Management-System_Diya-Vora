# Library Management System

This is a Python-based library management system that allows librarians to efficiently manage books, users, and borrowing transactions.

# Features

- Add, edit, and delete books
- Add, edit, and delete users
- Borrow and return books
- View book and user details
- Generate reports on borrowing history and availability of books

# Technologies Used

- Python 3.x
- Tkinter GUI toolkit
- SQLite database

# Setup Instructions

1. Clone this repository to your local machine.
2. Install Python 3.x on your system.
3. Install the required packages using the command `pip install -r requirements.txt`.
4. Run the program using the command `python library_management_system.py`.
